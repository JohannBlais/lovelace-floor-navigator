# Screenshots

This folder holds the marketing/documentation screenshots that the README
references. Owner is encouraged to drop the following images here :

- `hero.png` — main shot of the card on a 3D-rendered floor plan with several
  pastille icons visible. Width ≈ 1200 px.
- `transitions.gif` — short loop showing the `crossfade` / `slide` /
  `slide-scale` transitions side-by-side or sequentially.
- `overlay-toggle.gif` — short loop showing the toggle bar at the bottom and
  overlays appearing / disappearing on click.

Until these are added, the README and release pages render fine without them.
